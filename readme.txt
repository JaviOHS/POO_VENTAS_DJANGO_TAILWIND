Para instalar las dependencias del proyecto:
    pip install -r requirements.txt

Se debe crear un super usuario para acceder a los modulos de administracion.
Los usuarios creados en el formulario solo tienen acceso a la tienda.

Ejecutar para realizar cambios en 3 terminales: 
    cd proy_sales && py manage.py livereload
    cd proy_sales && py manage.py tailwind start
    cd proy_sales && py manage.py runserver
